# Utilização de IA e ajuda humana

1. A equipa definiu previamente o conceito VerdeVegan, as três tarefas principais, os fluxos e os protótipos de alta fidelidade.
2. A implementação Ionic/Angular foi acelerada com apoio de IA para estruturar páginas, services, modelos TypeScript, dados JSON e estilos responsivos.
3. A IA ajudou a interpretar o enunciado da Etapa 3 e a transformar os requisitos em funcionalidades verificáveis na aplicação.
4. A IA sugeriu a organização do código em services para menu, carrinho, encomendas, storage, strings, dispositivo e API externa.
5. A equipa deve validar manualmente os textos finais, executar a app em dispositivo físico e confirmar se o APK funciona antes da entrega.
6. A equipa deve acrescentar ao relatório o URL do repositório GitHub/GitLab quando este for criado.
