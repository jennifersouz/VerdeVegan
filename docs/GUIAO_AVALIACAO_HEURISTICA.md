# Guião para avaliação heurística

## Tarefa 1 - Efetuar pedido

1. Entrar na app.
2. Pesquisar "tofu" no menu.
3. Abrir o Bowl de Tofu Grelhado.
4. Adicionar ao carrinho.
5. Usar pontos, escolher pagamento MBWay e confirmar pedido.
6. Ver o ecrã de acompanhamento do pedido.

## Tarefa 2 - Personalizar refeição

1. Abrir um prato no menu.
2. Remover um ingrediente.
3. Adicionar um extra.
4. Escrever uma nota para o restaurante.
5. Adicionar ao carrinho e rever as alterações.

## Tarefa 3 - Repetir pedido, usar pontos e avaliar

1. Abrir o histórico.
2. Escolher um pedido anterior.
3. Repetir o pedido.
4. Confirmar no carrinho com pontos.
5. Abrir o pedido e deixar uma avaliação.

## Pontos a mostrar aos avaliadores

- Navegação com Angular Router e parâmetros.
- Dados vindos de JSON.
- Persistência com Ionic Storage.
- Reactive Forms.
- API externa OpenFoodFacts na página de detalhes.
- Capacitor Screen Orientation configurado para dispositivo físico.
